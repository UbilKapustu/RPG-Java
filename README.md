# RPG-Java
## Java Juniors` game for exp.
### Concept:
Мы создаем рпг-игру на джаве, в казахском сеттинге. 
Требования для ***MVP*** : 
- Наличие персонажей разных классов в игре.
- Наличие инвентаря, оружий, доспехов, зелий.
- Боевая пошаговая система.
- Наличие локаций, городов, подземелий.
- Возможность играть.
